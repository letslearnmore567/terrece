Build a full-stack web app called “Smart Terrace Farming System” for sustainable agriculture in hilly regions.

IMPORTANT CONSTRAINT:
This project must be optimized for Replit free version. Keep it lightweight, practical, and easy to run in a single Replit workspace. Do not overbuild unnecessary production-level features. Keep the important modules only, but make the app look complete and usable for a college/project demo.

PROJECT PURPOSE:
This app helps farmers monitor terrace farms using IoT sensor data and gives smart recommendations for irrigation, fertilizer timing, crop care, and farm management. The app should be farmer-friendly, simple, modern, clean, and mobile responsive.

CORE IDEA:
Sensors installed in the farm collect:
- soil moisture
- temperature
- humidity
- light intensity
- water level

An ESP32/NodeMCU device can send the data to the backend using API endpoints. The backend stores the data in a database, analyzes it using simple rule-based logic, and shows recommendations, alerts, and analytics in a dashboard.

This app must support both:
1. Demo mode with simulated/mock sensor data
2. Future IoT mode where ESP32 can send readings through REST API

TECH STACK:
- Frontend: React
- Styling: Tailwind CSS
- Backend: Node.js + Express
- Database: SQLite
- Charts: Recharts
- Authentication: simple JWT based login
- API: REST API

6. RECOMMENDATION ENGINE
Use simple rule-based logic only. Organize code in a way that AI/ML can be added later.

Rules:
- If soil moisture is below threshold, show “Irrigation needed”
- If temperature is high and humidity is low, show “Risk of crop stress”
- If water level is low, show “Check water source”
- If light is too low, show “Low sunlight detected”
- If light is too high, show “Excessive sunlight exposure”
- Based on crop type and growth stage, show fertilizer timing suggestion
- Show general crop care advice based on sensor conditions

Examples:
- “Soil moisture is critically low. Irrigation is recommended.”
- “Temperature is high and humidity is low. Crop stress risk detected.”
- “Water level is dropping. Check water storage or source.”
- “Current crop is in flowering stage. Nutrient support may be needed.”
